# Pharmacy Landing Page

A responsive landing page for a local pharmacy business, built with HTML, CSS, and JavaScript.

## Overview

This project was created for a pharmacy business that needed a clean, trustworthy, and easy-to-navigate online presence. The page focuses on presenting key business information clearly, including services, opening hours, hotline numbers, location details, and Google Maps directions.

## Features

- Responsive landing page layout
- Healthcare/pharmacy visual branding
- Service and product category sections
- Trust badges and business highlights
- Hotline call-to-action buttons
- Google Maps embed for directions
- Contact information and opening hours section
- Footer with legal/business information

## Tech Stack

- HTML
- CSS
- JavaScript
- Google Maps Embed
- Responsive Web Design
